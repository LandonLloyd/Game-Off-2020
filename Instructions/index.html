Instructions

Welcome to the game I made with the help from a few other contributors. In this game you are acting as the space invader on the left of the game board. You are invading our moon. NASA and a few other space exploration teams have put together a space army. They are firing bullets at you that you must jump over. If one hits you you lose. Good Luck and have fun!

How to play

The goal of the game is to jump over or crouch under the rocks. You can do this a few ways.

  1. Up Arrow - Pressing the up arrow will make you jump.
  2. Spacebar - Pressing the spacebar will also make you jump, this comes down to personal preference.
  3. Jump Button - Clicking the button that says Jump will also make you jump. This is more for mobile players.
  4. Down Arrow - Pressing the down arrow will make you duck under the bullets when they are to high to jump over.
  5. Crouch Button - Pressing the button that says Crouch will also make you crouch under bullets. Works better for mobile players.

Game Start

To start the game there are a couple of options

  1. Click the Start Game Button
  2. Refresh the browser

Doing either of these will automatically start the game so be quick!
